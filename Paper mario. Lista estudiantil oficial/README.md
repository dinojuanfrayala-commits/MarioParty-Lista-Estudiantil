# Mario Party - Lista Estudiantil 2026-2027

Sitio estático en HTML/CSS para la lista estudiantil inspirada en Mario Party.

## Contenido
- `mario_party_lista_estudiantil.html`: página principal del proyecto.
- `index.html`: redirige automáticamente a `mario_party_lista_estudiantil.html` para compatibilidad con GitHub Pages.
- Archivos de imagen incluidos en el mismo directorio.

## Cómo subir a GitHub
1. Crea un nuevo repositorio en GitHub.
2. Copia todos los archivos de esta carpeta al repositorio local.
3. Puedes inicializar Git o usar VS Code para crear el repositorio.
4. Sube los cambios a GitHub.

## Publicar con GitHub Pages
1. En el repositorio de GitHub, ve a `Settings` > `Pages`.
2. Elige la rama `main` o `master` y la carpeta `/root`.
3. Guarda y espera unos minutos.
4. El sitio quedará accesible en la URL proporcionada por GitHub.

## Nota
Si deseas que la página principal se abra directamente, usa `index.html` o navega a `mario_party_lista_estudiantil.html`.
